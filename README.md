# EasyCRUD Dockerized Application

A full-stack CRUD application using:

- Spring Boot (Backend)
- React + Vite (Frontend)
- MariaDB (Database)
- Docker & Docker Compose

## Provide ip in .env file (frontend)

## Run the application

```bash
docker compose up --build
```
For Mariadb

```bash
docker exec -it mariadb mysql -u easycrud -p
```
Password- easycrud
